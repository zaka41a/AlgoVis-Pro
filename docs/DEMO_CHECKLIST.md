# Demo Checklist

## Before Demo
- `npm run lint`
- `npm run test`
- `npm run build`
- `mvn -B test`

## Runtime Checks
- Backend URL reachable
- Frontend URL reachable
- Algorithm selector works
- Scenario selector works
- Player controls work (`Play`, `Pause`, `Next`, `Prev`, `Reset`, speed)
- Pseudocode highlight follows the current step
