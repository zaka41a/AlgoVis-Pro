# Presentation Guide (5-7 Minutes)

## Suggested Flow
1. Problem: algorithm understanding is hard with static code.
2. Solution: step-by-step visual execution with synced pseudocode.
3. Architecture: React frontend + Spring Boot API.
4. Live demo:
   - Sorting (Bubble or Quick)
   - KMP search
   - BFS/DFS traversal
5. Quality: tests and CI pipeline.
6. Conclusion: educational value and extension plan.

## Demo Checklist
- Backend running
- Frontend running
- API reachable
- One scenario tested for each algorithm family
